# 日照金山 · NLP 培訓師冥想引導 21 天自我訓練

一個離線可用的漸進式網頁應用程式（PWA），協助 NLP 培訓師循序鍛鍊「帶領學員冥想」的核心能力。視覺概念取自「日照金山」——晨光點亮金頂，象徵引導者以內在狀態（COACH State）點亮技巧。

## 功能

- **21 天旅程**：三週遞進課程（奠基 → 核心 → 整合），每天含訓練目標、核心練習、自我檢核與引導金句；可打卡、自動記錄進度環與連續天數（儲存於本機）。
- **練習工具**：
  - 呼吸節拍器（可調吸／屏／吐秒數，圓圈隨呼吸縮放，提醒吐氣時下指令）。
  - 引導練習計時器（含提示音）。
- **引導詞生成器**：融合米爾頓模式、同步與引導、次感官調整、心錨與未來投影，依主題（紓壓／焦慮／睡眠／自信／專注）× 隱喻場景 × 長度 × 技巧即時生成四階段引導詞，可複製、語音試聽、收藏。
- **我的收藏**：把生成的引導詞存於本機，隨時展開、複製、朗讀、刪除。
- **PWA**：可安裝到主畫面、離線使用。

## 技術

純前端靜態網站，無建置流程：

- `index.html` — 主應用程式（HTML / CSS / JS 內嵌）
- `manifest.json` — PWA manifest
- `sw.js` — Service Worker（離線快取）
- `icons/` — 應用程式圖示（日照金山）

## 本機執行

任一靜態伺服器即可，例如：

```bash
npx serve .
# 或
python -m http.server 8000
```

開啟 `http://localhost:8000/` 即可。

## 部署

本專案為靜態網站，可直接以 GitHub Pages（從分支根目錄）部署。
